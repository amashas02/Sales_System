# 🛒 Sales System (Java)

## 📌 Project Description
This is a Java-based Sales System developed to manage sales operations efficiently.  
The system allows users to handle products, process sales, and manage transactions with database support.

## 🚀 Features
- Add, update, and delete products
- Process customer sales
- Calculate total price
- Store and retrieve data using database
- Simple and user-friendly interface

## 🛠️ Technologies Used
- Java
- MySQL (SQL Workbench)
- Object-Oriented Programming (OOP)

## 🗄️ Database
The system uses **MySQL database managed via SQL Workbench** to store:
- Product details
- Sales records

## 📂 Project Structure
- `Main.java` – Entry point
- `Product.java` – Product model
- `Sales.java` – Sales logic
- `DBConnection.java` – Database connection handling

## ▶️ How to Run
1. Open the project in IntelliJ / Eclipse / NetBeans  
2. Set up MySQL database in SQL Workbench  
3. Update database credentials in the code  
4. Run `Main.java`

## 🎯 Purpose
This project was developed to understand Java application development with database integration and real-world sales system functionality.
